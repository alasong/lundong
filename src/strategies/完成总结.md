### 3. 实现策略绩效评估模块

**StrategyEvaluator** - 策略评估器

**评估指标**:
- 总收益 (total_return)
- 年化收益 (annual_return)
- 波动率 (volatility)
- 夏普比率 (sharpe)
- 最大回撤 (max_drawdown)
- 卡玛比率 (calmar)
- 胜率 (win_rate)
- 盈亏比 (profit_loss_ratio)
- 综合评分 (composite_score)

**使用示例**:
```python
from strategies.evaluator import StrategyEvaluator

evaluator = StrategyEvaluator()
metrics = evaluator.evaluate(portfolio_values, strategy_name="my_strategy")
print(f"夏普比率：{metrics['sharpe']:.2f}")
print(f"综合评分：{metrics['composite_score']:.1f}")
```

### 4. 添加动态权重调整功能

**DynamicWeightAllocator** - 动态权重分配器

**分配方法**:
- `equal`: 等权分配
- `sharpe`: 按夏普比率分配
- `return`: 按收益率分配
- `volatility`: 按波动率倒数分配 (低波策略)

**参数**:
```python
allocator = DynamicWeightAllocator(
    lookback_days=60,      # 回看 60 天
    rebalance_freq=5,      # 每 5 天再平衡
    min_weight=0.05,       # 最小权重 5%
    max_weight=0.40        # 最大权重 40%
)
```

**使用示例**:
```python
from strategies.dynamic_weights import DynamicWeightAllocator

allocator = DynamicWeightAllocator()

# 分配权重
weights = allocator.allocate_weights(
    strategy_returns={"strat1": returns1, "strat2": returns2},
    method="sharpe"
)

# 获取分配报告
report = allocator.get_allocation_report(strategy_returns, method="sharpe")
```
