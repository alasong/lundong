# 策略清单

## ✅ 已实现并注册 (5 个)

| 策略 | 类型 | 逻辑 | 参数 | 适用场景 |
|------|------|------|------|----------|
| **hot_rotation** | 热点轮动 | 板块热点+XGBoost 预测 | top_n_concepts, min_hotspot_score | 震荡市、结构性行情 |
| **momentum** | 动量策略 | 20 日涨幅 + 成交量突破 | momentum_window, min_momentum | 趋势市 |
| **mean_reversion** | 均值回归 | 布林带+RSI 超卖 | bb_period, rsi_oversold | 震荡市、超跌反弹 |
| **value** | 价值策略 | 低 PE/PB+高 ROE | max_pe, min_roe | 价值投资、长期持有 |
| **growth** | 成长策略 | 营收/利润高增长 | min_revenue_growth, min_profit_growth | 成长股投资 |

## 使用示例

### 单策略
```python
from strategies.register import *
from strategies.strategy_factory import StrategyFactory

# 创建策略
strategy = StrategyFactory.create_strategy("value")

# 生成信号
signals = strategy.generate_signals()
```

### 多策略组合
```python
from strategies.multi_strategy import MultiStrategyPortfolio

strategies = StrategyFactory.create_multiple_strategies({
    "hot_rotation": {"enabled": True, "weight": 0.4},
    "momentum": {"enabled": True, "weight": 0.3},
    "value": {"enabled": True, "weight": 0.3},
})

multi = MultiStrategyPortfolio(strategies=strategies)
merged = multi.generate_signals()
```
