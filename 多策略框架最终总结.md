## 📈 新增策略详解

### 10. SectorRotationStrategy - 行业轮动策略

**逻辑**：
- 计算各行业动量（20日涨幅）
- 选择强势行业TOP5
- 从强势行业中选股

**参数**：
```python
{
    "lookback_days": 20,        # 回看天数
    "min_sector_momentum": 0.05, # 最小行业动量5%
    "top_n_sectors": 5,         # 选择TOP5行业
    "stocks_per_sector": 3      # 每个行业选3只股票
}
```

**适用场景**：行业配置、结构性行情

### 11. DividendStrategy - 高股息策略

**逻辑**：
- 筛选股息率>3%的股票
- 连续分红年限>3年
- PE<20且ROE>8%

**参数**：
```python
{
    "min_dividend_yield": 0.03,  # 最小股息率3%
    "min_dividend_years": 3,     # 最小连续分红3年
    "max_pe": 20,                # 最大PE
    "min_roe": 0.08              # 最小ROE 8%
}
```

**适用场景**：防守配置、熊市、高波动市场

## 🎯 动态权重分配

**DynamicWeightAllocator** 支持4种分配方法：

1. **equal**：等权分配
2. **sharpe**：按夏普比率分配（推荐）
3. **return**：按收益率分配
4. **volatility**：按波动率倒数分配（低波策略）

**参数**：
```python
allocator = DynamicWeightAllocator(
    lookback_days=60,      # 回看60天
    rebalance_freq=5,      # 每5天再平衡
    min_weight=0.05,       # 最小权重5%
    max_weight=0.40        # 最大权重40%
)
```

## ⚠️ 注意事项

1. **数据质量**：回测依赖历史数据质量
2. **过拟合风险**：参数优化可能过拟合
3. **市场变化**：历史表现不代表未来
4. **交易成本**：回测未考虑手续费/滑点
5. **流动性**：小市值策略需注意流动性
6. **股息数据**：高股息策略需要准确的股息数据
7. **行业分类**：行业轮动策略需要准确的行业映射
