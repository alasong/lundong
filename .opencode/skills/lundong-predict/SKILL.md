# 预测执行专家

## 执行命令

```bash
python src/main.py --mode predict
```

## 执行流程

1. 数据验证（检查数据时效性）
2. 热点识别（板块热度评分）
3. 轮动分析（板块切换信号）
4. 模型预测（XGBoost 1d/5d/20d）
5. 组合优化（权重分配）
6. 生成报告

## 预期输出

- 热点板块排名
- 轮动信号
- 推荐股票列表
- 预期收益/风险评估
- 报告保存路径
