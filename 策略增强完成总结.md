### 6. 动态权重再平衡
```python

runner = StrategyRunner()
result = runner.dynamic_rebalance(
    strategy_config={
        "hot_rotation": {"enabled": True},
        "momentum": {"enabled": True},
        "value": {"enabled": True},
    },
    lookback_days=60,
    method="sharpe"
)
print(result["weights"])
```
